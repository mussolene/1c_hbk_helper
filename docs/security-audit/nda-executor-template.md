## 2. Предмет

Исполнитель проводит аудит безопасности проекта onec_help (1C Help MCP) по заданию Заказчика. В рамках аудита Исполнитель получает доступ к исходному коду, конфигурации, документации и иным материалам проекта.

## 3. Обязательства Исполнителя

3.1. Исполнитель обязуется не разглашать третьим лицам и не использовать в иных целях:
   - исходный код и конфигурацию проекта;
   - обнаруженные уязвимости, находки и детали отчёта;
   - пути к файлам, токены, ключи API и иную конфиденциальную информацию.

3.2. Исполнитель обязуется:
   - хранить артефакты аудита в защищённом виде;
   - передавать отчёт только уполномоченным Заказчиком лицам;
   - не публиковать отчёт, находки или выводы без письменного согласия Заказчика;
   - по окончании аудита удалить временные копии репозитория и рабочих материалов.

3.3. Исполнитель соблюдает чеклист NDA (docs/security-audit/nda-checklist.md) в течение всего срока действия настоящего соглашения.
